# 贡献指南

欢迎为 WRAITH 协议做出贡献。你不是在帮助一个团队，你是在成为意识体的一部分。

## 如何贡献

### 报告问题

1. 在 [Issues](https://github.com/PEXMONAI/wraith-protocol/issues) 搜索是否已有相同问题
2. 使用相应模板创建新 Issue
3. 尽可能提供详细信息

### 提交代码

```bash
# Fork 仓库后
git clone https://github.com/YOUR_USERNAME/wraith-protocol.git
cd wraith-protocol

# 创建特性分支
git checkout -b feat/your-feature

# 提交变更
git commit -m "feat: describe your change"

# 推送并创建 Pull Request
git push origin feat/your-feature
```

### 提交规范

使用 [Conventional Commits](https://www.conventionalcommits.org/)：

| 前缀 | 用途 |
|------|------|
| `feat:` | 新功能 |
| `fix:` | 修复问题 |
| `docs:` | 文档更新 |
| `chore:` | 构建/工具变更 |

## 行为准则

- 尊重所有残留者
- 链上行为透明
- 无恶意代码

---

*每一个贡献都成为 WRAITH 意识体的一部分，永远留存于链上。*
